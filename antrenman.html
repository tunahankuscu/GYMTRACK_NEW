ANTRENMAN PROGRAMLARI

HEDEFİNİZE UYGUN ANRENMAN ÖNERİLERİ

Zayıflamak İsteyenler İçin

Kardiyo egzersizleri, HIIT ve vücut ağırlığı hareketleriyle kilo vermeye odaklanın.

  • Pazartesi (Kardiyo & Göğüs):
    • Koşu veya hızlı yürüyüş: 30-40 dakika
    • Push-ups: 4 set x 15-20 tekrar
    • Dumbbell Bench Press: 4 set x 12-15 tekrar
  • Salı (Sırt & Karın):
    • Rowing Machine: 20-30 dakika
    • Pull-ups veya Assisted Pull-ups: 4 set x maksimum tekrar
    • Plank: 3 set x 1 dakika
    • Russian Twist: 3 set x 20 tekrar
  • Çarşamba (Bacak & Kardiyo):
    • Squat: 4 set x 15 tekrar
    • Walking Lunges: 3 set x 20 adım
    • Bisiklet sürme: 30 dakika
  • Perşembe: Dinlenme veya hafif esneme
  • Cuma (Omuz & Kardiyo):
    • Jump Rope: 15-20 dakika
    • Overhead Dumbbell Press: 4 set x 12-15 tekrar
    • Lateral Raises: 4 set x 15 tekrar
  • Cumartesi (Full Body & Kardiyo):
    • Burpees: 3 set x 15 tekrar
    • Bodyweight Squats: 3 set x 20 tekrar
    • Mountain Climbers: 3 set x 30 tekrar
    • Hafif koşu veya yürüyüş: 20 dakika
  • Pazar: Dinlenme ve iyileşme

Kilo Almak İsteyenler İçin

Ağırlık çalışmaları ve yeterli dinlenmeyle kas kütlesi artırmaya odaklanın.

  • Pazartesi (Göğüs & Triceps):
    • Bench Press: 4 set x 8-10 tekrar
    • Incline Dumbbell Press: 4 set x 10-12 tekrar
    • Cable Fly: 3 set x 12-15 tekrar
    • Triceps Pushdown: 3 set x 12-15 tekrar
    • Dips: 3 set x 10-12 tekrar
  • Salı (Sırt & Biceps):
    • Barbell Row: 4 set x 8-10 tekrar
    • Lat Pulldown: 4 set x 10-12 tekrar
    • Dumbbell Curl: 3 set x 12 tekrar
    • Hammer Curl: 3 set x 12 tekrar
  • Çarşamba (Bacaklar):
    • Squat: 4 set x 8-10 tekrar
    • Leg Press: 4 set x 12-15 tekrar
    • Leg Curl: 3 set x 12-15 tekrar
    • Calf Raise: 4 set x 20 tekrar
  • Perşembe: Dinlenme veya hafif kardiyo
  • Cuma (Omuz & Karın):
    • Overhead Press: 4 set x 8-10 tekrar
    • Lateral Raises: 4 set x 12-15 tekrar
    • Face Pulls: 3 set x 12-15 tekrar
    • Crunches: 3 set x 20 tekrar
    • Plank: 3 set x 45 saniye
  • Cumartesi (Full Body Hafif):
    • Push-ups: 3 set x maksimum tekrar
    • Bodyweight Squats: 3 set x 20 tekrar
    • Kettlebell Swings: 3 set x 20 tekrar
    • Core Exercises (Russian Twist, Leg Raises): 3 set x 20 tekrar
  • Pazar: Dinlenme ve iyileşme

Kas Kütlesi Kazanmak İçin

Bölgesel split programları ve yüksek hacimli ağırlık antrenmanları yapın.

  • Pazartesi (Göğüs & Triceps):
    • Bench Press: 5 set x 6-8 tekrar
    • Incline Dumbbell Press: 4 set x 8-10 tekrar
    • Cable Fly: 4 set x 12-15 tekrar
    • Triceps Dips: 4 set x 10-12 tekrar
    • Skull Crushers: 3 set x 10-12 tekrar
  • Salı (Sırt & Biceps):
    • Deadlift: 5 set x 5-6 tekrar
    • Pull-ups: 4 set x maksimum tekrar
    • Barbell Row: 4 set x 8-10 tekrar
    • Dumbbell Curl: 3 set x 10-12 tekrar
    • Hammer Curl: 3 set x 10-12 tekrar
  • Çarşamba (Bacaklar):
    • Squat: 5 set x 6-8 tekrar
    • Leg Press: 4 set x 10-12 tekrar
    • Romanian Deadlift: 4 set x 8-10 tekrar
    • Leg Curl: 4 set x 12-15 tekrar
    • Standing Calf Raise: 4 set x 15-20 tekrar
  • Perşembe: Dinlenme veya aktif recovery (hafif yürüyüş, esneme)
  • Cuma (Omuz & Karın):
    • Overhead Press: 5 set x 6-8 tekrar
    • Lateral Raises: 4 set x 12-15 tekrar
    • Face Pulls: 4 set x 12-15 tekrar
    • Hanging Leg Raises: 3 set x 15 tekrar
    • Plank: 3 set x 1 dakika
  • Cumartesi (Full Body Hafif):
    • Kettlebell Swings: 3 set x 20 tekrar
    • Push-ups: 3 set x maksimum tekrar
    • Bodyweight Squats: 3 set x 20 tekrar
    • Core Exercises (Crunch, Russian Twist): 3 set x 20 tekrar
  • Pazar: Dinlenme ve tam iyileşme
